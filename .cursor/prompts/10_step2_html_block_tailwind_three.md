# Step 2：HTMLブロックで Tailwind CDN + three.js を動かす（日本語・丁寧語で回答してください）

## 目的
固定ページ「Playground」に、カスタムHTMLブロックを追加し、以下を実現してください。
- Tailwind CSS を CDN で読み込む
- three.js を ES Modules で読み込み、canvas 上でアニメーション表示する
- Tailwind で文字を重ねてレイアウトする

## 言語
- 回答はすべて日本語で、丁寧語（です・ます調）で記述してください。

## 要件
- `<script type="module">` を使用し、`three.module.js` を unpkg から import してください。
- 「貼り付けるだけ」で動く、単一のHTMLスニペットとして提示してください。
- リサイズ対応を必ず含めてください。
- `devicePixelRatio` は上限を設けてください（例：2）。
- 最初は外部アセット（画像・モデル）なしで構いません。

## 出力
- WordPress管理画面での操作手順（どのブロックに貼るか）も併記してください。
- そのままコピペ可能なHTMLを提示してください。
